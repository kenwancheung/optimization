using JuMP
using GLPKMathProgInterface

myModel = Model(solver=GLPKSolverLP())
@variable(myModel, x1 >=0)
@variable(myModel, x2 >= 0)
@constraint(myModel, -1*x1 + x2 <= 1)
@constraint(myModel, -0.5*x1 + x2 <= 2)
@constraint(myModel, x1 + x2 <= 9)
@constraint(myModel, 0.5*x1 + x2 <= 6)
@objective(myModel, Max, x2)
myModel

D. Justify the optimal solution

Given the linear constraints, we are trying to find the maximum feasible solution for x2, which from the solver we know occurs at x2=4. From the graph above, you can see that the optimal solution occurs at one of the corners of the polyedron, where the objective function and constraints overlap x2=4 (and x1=4), which is the solution we found using the solver.
